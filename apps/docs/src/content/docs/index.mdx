## What is AlbertPlus?

AlbertPlus is a powerful, open-source platform designed to enhance the course registration experience for NYU students. It provides a modern, intuitive interface and a suite of tools to help students plan their academic journey, build schedules, and navigate the complexities of course selection. This project is developed and maintained by the vibrant community at Tech@NYU.
